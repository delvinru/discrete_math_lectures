## Сборка проекта
Для сборки проекта вам необходимо иметь пакет `texlive-full`. Установить его можно следующим образом:

### Ubuntu
```bash
sudo apt update
sudo apt install texlive-full
```

### Arch Linux
```bash
sudo pacman -Syyu
sudo pacman -S texlive-most texlive-lang
```

Сборка проекта:
```bash
make
```
